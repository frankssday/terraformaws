# terraformaws

## Prerequites
- Python3 (3.6 or above)
- Terraform
- Python package Boto3


## Run the example
```
$ git clone https://github.com/frankssday/terraformaws.git
$ cd terraformaws/
$ pip3 install -r requirements
$ ls
data.json  README.md  requirements  s3.tf  upload2s3.py
$ terraform init
$ terraform apply
$ python3 upload2s3.py
```
